# LoopTimmy
A minimalist, looping timer designed for relentless consistency.

## Description
LoopTimmy is a "Consistency Machine" that automatically restarts your countdown the moment it hits zero. It solves the friction of manual resets, ensuring your habits and deep-work intervals stay on track without interruption.

## Features
- Auto-Looping: Restarts immediately upon completion.
- Resilient Logic: Uses system clock timestamps to prevent background freezing on mobile.
- Minimalist UI: High-contrast, mobile-first design with rounded controls.
- Persistent Alerts: Audio beeps to signal cycle completions.

## How to Use
- Follow this link -> [LoopTimmy](https://l-zia-l.github.io/looptimmy/)
-Set your desired interval using the + and - buttons.
-Press Play to begin the infinite loop.
-Grant notification permissions when prompted to receive alerts.
-Press Pause or Reset to stop the machine.

## Technical Stack
-Frontend: HTML5, CSS3 (Flexbox/Grid), JavaScript.
-Clock Logic: Date.now() synchronization for mobile battery-saver compatibility.
-Hosting: Securely served via GitHub Pages (HTTPS).
